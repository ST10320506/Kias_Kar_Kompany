<a name="usage"></a>
## Usage
- ## Kias Kar Kompany
- To use this program, you would need to open the project file in Visual Studio 2026. Make sure the file ending in .csproj is open.
- Click the green play button to start debugging and run the program on IIS Express, and it eventually opens a web application such as Chrome.
- You will be taken to the landing page, which is the home page with a navigation bar at the top of the page.
- Use the navigation bar or buttons to navigate to each page (Home, Manufacturer, Vehicle, Owner, or Privacy).
- Each item on the table has a CRUD function to either edit, view details, or delete the item entered on the table.
- Each page has an option to create, edit, delete, or view details of an owner, manufacturer, or vehicle.
- This application also has an image URL placeholder to store and display images of each vehicle.
- To exit the application, you can close the web application (Chrome) by clicking on the 'x', which will stop the program from running.

<a name="features"></a>
## Features:
- ## Kias Kar Kompany
- **Feature 01**: This application is an ASP.NET Core MVC project, so it runs on a web application software such as Chrome.
- **Feature 02**: This application has not used scaffolding, which means the CRUD functions for each model class were coded manually. 
- **Feature 03**: This application has migrations to create and update the local database.
- **Feature 04**: This application has a navigation bar functionality on the home page to navigate to each page.
- **Feature 05**: This application has an option to create a new owner, vehicle, and manufacturer on each page.
- **Feature 06**: This application handles an image URL for vehicles.
- **Feature 07**: This application allows items to be created, saved, and displayed on the web page and onto a local database.
- **Feature 08**: This application has CRUD (Create, Read, Update and Delete) functionality for each item on the table.
- **Feature 09**: This application shows data persistence as the data entered on the web application is saved onto the database and appears when the application is closed and opened again.

- **Project stage**: This project is in the beginning stages, where we have developed the basic Controllers, CRUD and Views for the application.
- **Future advancement**: The future advancements of this application can be to add CSS styling.
